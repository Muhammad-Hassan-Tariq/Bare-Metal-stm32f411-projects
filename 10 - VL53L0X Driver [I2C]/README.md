# 10 - VL53L0X Time of Flight Sensor Driver

## Overview
- This project demonstrates a **Distance measuring using TOF Sensor VL53L0X** on the STM32F411 MCU.

## Hardware Setup  
- 1x USB-TTL Adaptor
- picocom Software
- 1x Active Buzzer
- 1x LED
- 1x VL53L0X

## Working  
- Using `sudo picocom -b 115200 /dev/ttyUSB0` Transmission can be seen in terminal.
